# Memory bank — ppsite

Dit is de **contextbank** voor het Next.js-project **ppsite** (Poule & Poulette). Houd ze bij bij elke merkbare beslissing of feature, zodat vervolggesprekken dezelfde lijn kunnen volgen.

## Bestanden

| Bestand | Doel |
|--------|------|
| [progress.md](./progress.md) | Chronologisch: wat gebouwd is, waar het staat, open punten |
| [../DESIGN.md](../DESIGN.md) | Design system in [awesome-design-md](https://github.com/VoltAgent/awesome-design-md)-stijl (tokens, type, componenten, agent-prompts) |
| [../AGENTS.md](../AGENTS.md) | Korte instructie voor coding agents |

## Workflow voor agents

1. Lees **progress.md** voor de huidige stand en beslissingen.
2. Lees **DESIGN.md** vóór UI-, kleur- of typografie-wijzigingen.
3. Na een sessie: **progress.md** bijwerken (datum, wijziging, bestanden).
